## 프로젝트 디자인 요구사항

## UI/UX 원칙
- 접근성: WCAG 2.1 준수
- 직관적인 네비게이션과 피드백
- 사용자 중심의 디자인

## 색상 팔레트
- Primary: #3B82F6
- Secondary: #64748B
- Accent: #F59E0B
- Background: #FFFFFF
- Text: #1F2937

## 폰트
- 기본 폰트: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif
- 제목 폰트: Bold (700)
- 본문 폰트: Regular (400)

## 반응형 디자인
- Mobile: <= 768px
- Tablet: 769px - 991px
- Desktop: >= 992px

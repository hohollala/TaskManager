# newProject

## 설명
새 프로젝트 요구사항을 생성합니다

## 카테고리
프로젝트 관리

## 사용법
```
newProject [프로젝트 설정]
```

## 예시
- `newProject`
- `newProject --purpose '할일 관리 앱'`
- `newProject --features '사용자 로그인, 할일 CRUD'`
